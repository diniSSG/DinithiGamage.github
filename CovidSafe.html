CovidSafe

description,

CovidSafe is an innovative application designed to facilitate safe entry into public places by implementing QR code scanning for COVID-19 exposure tracking. The application enables individuals to scan QR codes displayed at various locations, providing real-time information on potential COVID-19 exposure and supporting contact tracing efforts.

Key Features:

  • QR Code Scanning: Users can scan QR codes displayed at entry points of public places such as restaurants, shops, and offices using their mobile devices. This process logs the user's visit to the location securely within the application.
  • Exposure Notification: CovidSafe alerts users if they have visited a location where a confirmed COVID-19 case was reported. This feature helps individuals monitor their exposure risk and take necessary precautions.
  • Location-based Data Collection: The application collects anonymized location data through QR code scans to support contact tracing efforts. This data is used to identify potential COVID-19 exposure hotspots and notify users accordingly.
  • User Dashboard: Provide a personalized dashboard where users can view their booking history, upcoming trips, and manage their account details.
  • Privacy Protection: CovidSafe prioritizes user privacy by anonymizing and securely storing location data. Personal information is protected in accordance with data privacy regulations.
  • Real-time Updates: The application provides real-time updates on COVID-19 cases and government guidelines, empowering users to make informed decisions about their activities and travel.

Technologies Used

  • asp.net web framework 4.5 / web APIs
  • SQL Server
  • Postman
  • HTML / CSS & JavaScript
  • QR Code Integration / Android & Geolocation Services
