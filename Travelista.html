Travelista

description,

Travelista Flight Reservation System is an endeavour to simulate a basic airline reservation system that will fulfil all the administration, management and payment, services and system requirements. This is a distribution system design for booking desired airline from anywhere.

The passengers can check flight details via integrated website and send their details to the centralized system to confirm the reservation and settle down payments. The website is provide options for passengers to view all the information, destinations, durations ,different flights and their availability along with timing , dates, etc…Also passengers can get updates about discounts and special offers provided to them

First, to use our system, customer need to create a user account on website and be register into the system. The website displays all the details required to make a reservation. After login, customer is enable to search and look for details of airlines in order to choose a particular flight, based on the details but it does not allow user to book a ticket, modify or cancel a particular reservation.

For schedule a booking, customer need to fill all the required details such as name, passport number, departure date ,arrival date etc…, in a format which is use to input customer data into the system database .After filling the form the staff will contact the customer to conform their reservation and continue the payment process via the system. In addition, to cancel or modify a reservation customer should contact the staff.

Technologies Used

  • REST API & postman
  • HTML / CSS3
  • asp.net &.net core
  • SQL Server
