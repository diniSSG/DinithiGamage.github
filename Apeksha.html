Apeksha

description,

For my final year individual project, I undertook the development of a cancer disease prediction system utilizing advanced data mining techniques. The objective was to create an application that could accurately predict the likelihood of cancer based on a set of input parameters derived from clinical data. To achieve this, I leveraged a clinical database containing extensive patient records. The system, developed as a hybrid application comprising both a mobile app and a web application, aimed to provide accessible and user-friendly access to predictive analytics. The heart of the system lies in its classification model, which was trained using a dataset split into two parts: one for training the model and the other for testing its accuracy. Employing three distinct algorithms—K-Nearest Neighbors (KNN), Logistic Regression, and Random Forest Classifier—the classification model was meticulously crafted to ensure optimal performance and robustness. Through rigorous testing and validation, the system demonstrated a higher accuracy in predicting cancer diagnoses, thereby offering a valuable tool for early detection and intervention. This project not only showcased my proficiency in data mining and predictive analytics but also underscored my commitment to leveraging technology for the betterment of healthcare outcomes.

Key Features:

  • User to register login and create a personalized account.
  • User to input patient’s FNA scan records to predict cancer risk
  • User to add medical records and create a medical log and get a notification reminder
  • Monitor how treatments and therapies make you feel
  • To share user’s thoughts and experiences with other cancer survivors
  • To contact their desire consultations.

Technologies Used

  • Flutter / FireBase
  • HTML / CSS3 & JavaScript
  • Python Anaconda / Machine Learning
  • JSON
